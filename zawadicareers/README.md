# Zawadi Careers - Career Development Platform

Zawadi Careers is a comprehensive career development and opportunity platform built to help Africans discover their gifts, develop relevant, in-demand skills, and connect to meaningful career opportunities.

## Features

### For Users
- **Multi-page Blog Reading**: Blogs broken into sections with next/previous navigation
- **No Login Required**: Users can read, comment, and subscribe without accounts
- **Comment System**: Comment with name/email (no login required)
- **Newsletter Subscription**: Pop-up reminders and subscription forms
- **Mobile-First Design**: Responsive design with mobile bottom navigation
- **Category-Based Content**: 7 career-focused categories
- **Like & Share**: Social engagement features
- **Search Functionality**: Search blogs by title and content

### For Admins
- **Admin Dashboard**: Analytics and user tracking
- **Blog Management**: Create, edit, and publish blogs
- **Media Management**: Upload images and videos
- **Comment Moderation**: Approve/reject comments
- **Subscriber Management**: View and manage newsletter subscribers
- **Analytics**: Track views, reads, and engagement

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Database**: Supabase (PostgreSQL)
- **Styling**: Custom CSS with DeepSeek AI blue color scheme
- **Icons**: Font Awesome 6
- **Fonts**: Poppins & Inter from Google Fonts
- **Rich Text Editor**: Quill.js (for admin blog editor)
