# Zawadi Careers Web Application

A comprehensive career development platform for Africans, providing career insights, skill development resources, and job opportunities.

## Features

### User-Facing Features
1. **Responsive Design** - Mobile-first approach with mobile bottom navigation
2. **Blog System** - Categorized career articles with paginated reading experience
3. **Newsletter Subscription** - Popup and inline subscription forms
4. **Comment System** - Anonymous commenting with name/email
5. **Like System** - Blog post liking functionality
6. **Categories** - Organized content by career stages and needs
7. **Search & Filter** - Blog search and category filtering

### Admin Features
1. **Authentication** - Secure admin login/signup with Supabase Auth
2. **Dashboard** - Analytics and platform overview
3. **Blog Management** - Create, edit, and publish blog posts
4. **Rich Text Editor** - WYSIWYG editor with image/video support
5. **Media Management** - Upload and manage blog images
6. **Analytics** - View traffic and engagement statistics
7. **Comment Management** - Moderate user comments
